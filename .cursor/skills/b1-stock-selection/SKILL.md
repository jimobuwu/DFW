# 波段选股策略技能（B1 + B2 + 砖型图战法）

## 策略总览

本项目的核心预选策略体系，目标是在 A 股中筛选出 **处于波段启动初期、具备主力建仓迹象** 的股票。完整流水线：

```
拉取日线数据 → 流动性池筛选 → B1/B2/砖型图 量化初选 → 导出 K 线图 → AI 图表复评 → 推荐输出
```

三大策略的定位与适配：

| 策略 | 平均涨幅 | 适合人群 | 核心特征 |
|------|----------|----------|----------|
| B1 | ~2.14% | 求稳新手，容错高 | KDJ 超跌 + 知行线多头 |
| B2 | ~2.46% | 有经验者，需结合 B1 | B1 后缩量回调的倍量反包 |
| 砖型图 | ~2.87% | 能承受高波动者 | 红绿砖转换的爆发买点 |

## 一、双线战法核心逻辑

双线战法是 B1/B2 策略的理论基础，源自知行合一交易体系。

### 1. 什么是双线

- **白线（短期趋势线 / zxdq）**：double-EWM 平滑线 `ewm(span=10).ewm(span=10)`，股价短期快速上涨时一般不会跌破白线，可以看做流畅波段的"牵牛绳"
- **黄线（知行多空线 / zxdkx）**：四均线平均 `(MA_m1 + MA_m2 + MA_m3 + MA_m4) / 4`，主力是否入场的标志性参考线，也是极限洗盘的极限位，也是大哥控盘的成本线

### 2. 双线使用逻辑

- **右侧交易只做白线在黄线上的票**
- **短期放量站上白线**，再结合形态判断是否具备上涨爆发力；连续拉升后如果回踩可看做第一支撑，如果破掉即宣告**短期趋势大概率走坏**
- **黄线（大哥线）线下无右侧交易价值**；连续放量站上即说明大哥入场，之后考虑 B1 跟进；连续上涨后跌破，次日无法收回则宣告**中期趋势大概率走坏**
- **横盘震荡区间白线和黄线操作优势不显著**，切换成顶部压力线与底部支撑线来评估买卖位置

### 3. 双线使用核心玩法

- **白线金叉黄线**：上涨趋势确立概率大，等回踩 B1 可入场
- **白线死叉黄线**：最后的离场时机，走错也要走
- **白线与黄线要死叉而未死叉时**：N 型结构不破，配合 J 值负值，反而是一个极限买点（需设好止损只输一根）
- **N 型上涨趋势中，白线与黄线区间的位置**：都是容错率相对较高的买入区
- **放量金叉后缩量回踩黄线**：交易价值非常大，往往是连续拉升前的最后震仓动作

### 4. 循环操作法

- B1 买入 → 横盘或温和上涨就拿住 → B1+白黄区间加仓 → 加速后就开始放飞
- 跌破白线就卖出 → 继续观察白黄区间建仓价值 → 重新开始循环 B1 买入

## 二、B1 策略四重过滤条件

B1Selector 由 4 个独立 Filter 组成，全部通过才选入：

### 1. KDJ 分位过滤（KDJQuantileFilter）

捕捉 J 值处于历史低位的股票（超跌信号）。

- **条件**：`J < j_threshold` OR `J <= 历史累积 j_q_threshold 分位`
- **默认参数**：`j_threshold=15.0`, `j_q_threshold=0.10`, `kdj_n=9`
- **向量化实现**：使用 `expanding().quantile()` 保证无未来信息泄漏
- **Numba 加速**：KDJ 递推 `K[i] = 2/3 * K[i-1] + 1/3 * RSV[i]`，`J = 3K - 2D`

### 2. 知行线条件过滤（ZXConditionFilter）

确认股价在长期均线之上，趋势向好。

- **条件**：`close > zxdkx`（价格在均线上）且 `zxdq > zxdkx`（快线在均线上）
- **zxdq**：double-EWM 平滑线，`ewm(span=10).ewm(span=10)`
- **zxdkx**：四均线平均 `(MA_m1 + MA_m2 + MA_m3 + MA_m4) / 4`
- **默认参数**：`zx_m1=14, zx_m2=28, zx_m3=57, zx_m4=114`

### 3. 周线均线多头排列过滤（WeeklyMABullFilter）

确认中长期趋势健康。

- **条件**：周线 `MA_short > MA_mid > MA_long`
- **默认参数**：`wma_short=10, wma_mid=20, wma_long=30`（周线周期）
- **实现**：日线 → ISO 周分组 → 周线收盘价 → 滚动均线 → ffill 回日线

### 4. 成交量最大日非阴线过滤（MaxVolNotBearishFilter）

排除主力出货迹象。

- **条件**：过去 n 日内成交量最大的那天 `close >= open`（非阴线）
- **默认参数**：`n=20`
- **Numba 加速**：O(N×n) 滚动窗口

## 三、B2 策略（B1 后的二次买点）

B2 是 B1 之后的**倍量柱反包买点**，属于趋势确认后的加仓/补仓机会。B2 经典图形的核心模式：

### B2 买点识别规则

1. **前提**：股票必须先经历过 B1 买点（KDJ 低位 + 知行线多头）
2. **缩量回调**：B1 后股价出现缩量回调（成交量逐步萎缩），回调踩到黄线附近
3. **倍量反包**：回调后出现一根**放量阳线**，成交量 ≥ 前一日的 1.3~2 倍（理想为倍量），且阳线实体覆盖前一日阴线
4. **K 线形态要求**：
   - 缩量回踩黄线后的阳线，只要放量 30% 就行
   - 横盘后的超级倍量柱是强买点
   - 大长下影的放量阳线 B2 需警惕（反面案例）

### B2 经典形态总结（来自 22 个实战案例）

| 形态 | 描述 | 典型案例 |
|------|------|----------|
| 缩量回调倍量反包 | 标准 B2，回调后放量阳线反包 | 美迪西、德龙激光 |
| 横盘后倍量柱 | 横盘整理后突然出现超级倍量线 | 路桥信息、指南针 |
| 多次击穿后放量 | 多次击穿黄线后阳线悄悄放量 | 安德利 |
| 缩量踩黄线反包 | 最完美的 B2 图形 | 美迪西、三峡水利 |
| 关键K + 缩量下跌 + 倍量 | 复合条件触发 | 京北方 |
| 红肥绿瘦 + 倍量 | 横盘中阳线量大于阴线 | 指南针 |

### B2 量化条件（建议实现）

```python
# B2 核心条件
1. 近 N 日内存在 B1 买点（lookback=30）
2. B1 后出现缩量回调：volume 连续 >= 2 日递减
3. 回调低点接近黄线：close 在 zxdkx * [0.97, 1.03] 区间
4. 今日出现放量阳线：
   - close > open（阳线）
   - volume >= prev_volume * 1.3（放量 30%+）
   - close >= prev_close（反包或上涨）
5. 非涨停：涨幅 < 9.8%
```

### B2 反面案例警示

- 大长下影的放量阳线：信号不可靠
- 前期已大幅拉升（涨幅>50%）：B2 失效概率高
- 堆量 + 长上下影 + 犬牙交错 + 无序震荡的"狗啃式"图形：直接放弃

## 四、砖型图策略

### 1. 砖型图基础概念

- **砖**：指单根 K 线收盘后经砖型图算法计算形成的实体值
- **红砖**（brick > 0）：阳线方向，多头
- **绿砖**（brick < 0）：阴线方向，空头
- **大小标准**：创业板 20 厘米涨跌幅下，长阳标准需 ≥ 10 厘米（涨幅除以 2 判断）
- **核心规则**：以砖型图颜色为准，非 K 线视觉；阳线实际跌幅 2% 仍属有效红砖
- **算法价值**：砖型图设计旨在规避 K 线欺骗，主力难以伪造

### 2. 砖型图五重过滤（BrickChartSelector 现有实现）

1. **涨幅限制**：今日涨幅 < `daily_return_threshold`（排除涨停追高）
2. **红柱确认**：今日 brick > 0
3. **绿柱翻红**：昨日 brick < 0
4. **力度判断**：`今日红柱 >= brick_growth_ratio × |昨日绿柱|`
5. **连续绿柱**：红柱前至少 `min_prior_green_bars` 根绿柱

### 3. 砖型图三大买入定式（新增知识）

#### 定式一：N 型起跳

- **形态**：K 线呈现"低点 → 高点 → 低点 → 起跳"的 N 型结构，起跳位置必须为红砖
- **有效判定**：
  - 第一块红砖无意义，需观察是否处于 N 型结构确认的低点位置
  - 配合砖型图大小与上影线状态
  - 买入后第二块红砖必须向上突破，若再横盘即失效
- **识别要素**：位置（N 型低点） + 砖大（红砖实体足够） + 形态（无大上影线）

#### 定式二：横盘突破

- **形态**：价格在窄幅区间内横盘整理后出现红砖启动
- **关键点**：
  - 横盘可为"大横盘加小横盘"，关键在于"有横盘动作的突破"
  - 横盘后首根红砖即为买点，但需警惕"假突破"
  - 成功标志是突破后连续红砖，失败则迅速离场
- **强化信号**：突破 K 线为"暴力K + 关键K"，且覆盖前期套牢盘

#### 定式三：上升波段延续

- **形态**：上涨趋势中无显著回调，K 线表现为"红→绿→红"快速切换
- **区别**：中间仅 1-2 根 K 线，非 N 型亦非横盘，属趋势惯性延续
- **注意**：
  - 需区别于"连续红砖无调整"的高风险形态
  - 必须确认前高突破
  - 回调形成 1-2-3-4 标准砖序列时，第四块红砖为"放飞"信号

### 4. 四块砖止盈纪律（核心卖出规则）

这是砖型图策略最重要的风控纪律：

```
数四块红砖 → 第四块后必须减仓或清仓 → 绿砖出现即离场
```

- **铁律**：数满四块红砖后须减仓或清仓
- **反转处理**：绿砖出现即离场；若绿砖后接红砖，新一轮四块砖开始计数
- **收盘前纪律**：收盘前三分钟红翻绿，必须当日离场
- **次日应对**：开盘后 9:33-9:37 未大幅拉升即走；若高开 4% 以上可持有
- **循环结构**：1234 放飞 → 2234 → 循环结构，严格数砖

### 5. 砖型图形态红线（排除条件）

- **大苗子（大上影线）**：上影线长度显著超过实体，一律排除
- **跳空缺口**：不参与
- **锤子与棒子区分**：锤子（下影长、实体小）vs 棒子（实体大、上下影均短），用法不同
- **小上影线**：允许存在，属合规形态

### 6. 附加条件（现有实现）

- `close < zxdq × zxdq_ratio`：价格未过度偏离知行线
- `zxdq > zxdkx`：知行线多头
- 周线均线多头排列
- 必须在**多头区间（白线/黄线上方）**操作

## 五、高位与低位的相对性

### 核心原则

- **没有绝对高低，只有相对位置**：高位或低位取决于所选观察周期
- **黄线代表市场均值**，偏离黄线较远即界定为高位/低位
- **动态视角**：同一位置，买入时为低位，止损后即成高位

### 黄线交易法则

- **黄线跌破即熄火信号**：股价跌破黄线，大概率趋势终结，须果断离场
- **成本优势决定持仓底气**：低位买入者可承受回调，高位买入者无成本优势
- 即便参与黄线位置，也须右侧交易，设好止损

## 六、图形识别禁区

以下图形应直接放弃，不做 B1/B2/砖型图：

1. **堆量 + 长上下影 + 犬牙交错 + 无序震荡**的"狗啃式"图形
2. **"三波"非"三门重炮"**：若第三波后未快速突破，极可能是主力出货诱多
3. **连续红砖无调整**：无本之木，高风险
4. **白线距离黄线太远**：成功率低

## 七、流动性池构建

- 使用 `TopTurnoverPoolBuilder`
- 按每日滚动成交额（`turnover_n`，窗口 `n_turnover_days=43` 日）跨市场排名
- 取 top_m（默认 5000）只股票进入候选池
- `turnover = (open + close) / 2 × volume`

## 八、数据预处理流水线

`MarketDataPreparer` 执行以下步骤（多进程并行）：

1. 列名统一小写，日期转 datetime，排序
2. warmup 切片（确保均线有足够历史数据）
3. 计算 `turnover_n`
4. 设置 DatetimeIndex
5. 调用 `selector.prepare_df()` 预计算所有指标列

## 九、AI 复评评分维度

量化初选后，导出 K 线图交给 AI（智谱 GLM-4V-Flash）做视觉评分：

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 趋势结构 | 0.20 | 均线多头排列、上拐、交叉 |
| 价格位置 | 0.20 | 中低位突破 > 接近前高 > 高位过热 |
| 量价行为 | 0.30 | 上涨放量+回调缩量最佳，放量大阴最差 |
| 前期建仓异动 | 0.30 | 异常放量中大阳+突破平台最佳 |

判定：`total_score >= 4.0` → PASS，`3.2 ~ 4.0` → WATCH，`< 3.2` → FAIL

## 十、调参指南

```yaml
b1:
  enabled: true
  zx_m1: 14         # 知行线短周期（越小越敏感）
  zx_m2: 28
  zx_m3: 57
```

- 想选更多股票：提高 `j_threshold` 或 `j_q_threshold`
- 想更严格：降低阈值，或调大 `zx_m4`
- `top_m` 越大，流动性池越宽，覆盖更多小盘股
- 砖型图 `brick_growth_ratio` 越高，要求红柱力度越强

## 十二、交易心态与纪律

来自实战教学的核心提醒：

1. **"拿住不动"陷阱**：70% 浮盈常被回吐至 30% 甚至亏损，必须有止盈纪律
2. **市场唯一确定的是"应对"而非预测**：买点是情况，卖点是变化后的应对
3. **所有战法本质为"守株待兔"**：等待形态达标而非主动预测
4. **单笔亏损严格受限**：避免"一把亏大"，确保始终处于"游走状态"
5. **若经历完整多头区间仍无法盈利**：该方法不适用，应果断退出
